# Evolution Icons Folder

Place your pixel art PNG icons here for evolution conditions.

## Required Icons (16x16 or 24x24 pixels recommended):

### Time-based
- `day.png` - Sun icon for daytime evolutions
- `night.png` - Moon/star icon for nighttime evolutions
- `dusk.png` - Sunset icon for dusk evolutions

### Weather & Environment
- `rain.png` - Rain cloud icon
- `location.png` - Map pin or location marker

### Gender
- `male.png` - Male symbol (Mars symbol ♂)
- `female.png` - Female symbol (Venus symbol ♀)

### Friendship & Stats
- `happiness.png` - Heart or smiley face
- `beauty.png` - Flower or sparkle
- `affection.png` - Double heart

### Battle & Moves
- `move.png` - Attack/move icon
- `party.png` - Multiple Pokéballs icon

### Special Conditions
- `upside-down.png` - Arrow or device rotating

## Design Tips:
- Use pixel art style to match Pokémon aesthetics
- Keep icons simple and recognizable at small sizes
- Use colors that stand out against the evolution chain background
- Consider transparency for better integration

## Example Sources:
- Design your own in Aseprite or similar pixel art tools
- Use Pokémon game sprites as reference
- Check game UI for icon inspiration
